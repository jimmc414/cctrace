Claude Code Session Export - Usage Examples
==========================================

1. Basic Export (Slash Command)
-------------------------------
In Claude Code, simply type:
/user:export

Output:
🔍 Looking for Claude Code sessions in: /mnt/c/python/myproject
📂 Found 1 session(s) for this project
📤 Exporting session: a1b2c3d4-5678-90ab-cdef-123456789012
✅ Session exported successfully!
📁 Output directory: /home/user/claude_sessions/exports/2025-07-02_14-30-00_a1b2c3d4

Files created:
  - session_info.json
  - conversation_full.md
  - raw_messages.jsonl
  - summary.txt


2. Multiple Active Sessions
---------------------------
When multiple sessions are active:

$ python3 ~/claude_sessions/export_claude_session.py

🔍 Looking for Claude Code sessions in: /mnt/c/python/myproject
📂 Found 3 session(s) for this project
🔍 Found 2 active sessions:
  1. a1b2c3d4... (modified 45s ago)
  2. e5f6g7h8... (modified 120s ago)

🎯 Attempting to identify current session...
📍 Current Claude Code PID: 12345
✓ Session a1b2c3d4... was modified after marker creation
✅ Successfully identified current session: a1b2c3d4-5678-90ab-cdef-123456789012

📤 Exporting session: a1b2c3d4-5678-90ab-cdef-123456789012
✅ Session exported successfully!


3. Export Specific Session
--------------------------
$ python3 ~/claude_sessions/export_claude_session.py --session-id e5f6g7h8-1234-56ab-cdef-098765432109

🔍 Looking for Claude Code sessions in: /mnt/c/python/myproject
📂 Found 3 session(s) for this project
📤 Exporting session: e5f6g7h8-1234-56ab-cdef-098765432109
✅ Session exported successfully!


4. Custom Output Directory
--------------------------
$ python3 ~/claude_sessions/export_claude_session.py --output-dir /backup/claude-sessions

🔍 Looking for Claude Code sessions in: /mnt/c/python/myproject
📂 Found 1 session(s) for this project
📤 Exporting session: a1b2c3d4-5678-90ab-cdef-123456789012
✅ Session exported successfully!
📁 Output directory: /backup/claude-sessions/2025-07-02_14-30-00_a1b2c3d4

6. Help Information
-------------------
$ python3 ~/claude_sessions/export_claude_session.py --help

usage: export_claude_session.py [-h] [--session-id SESSION_ID]
                               [--output-dir OUTPUT_DIR]
                               [--format {md,html,all}]
                               [--max-age MAX_AGE]

Export Claude Code session

optional arguments:
  -h, --help            show this help message and exit
  --session-id SESSION_ID
                        Specific session ID to export
  --output-dir OUTPUT_DIR
                        Custom output directory
  --format {md,html,all}
                        Output format (default: md)
  --max-age MAX_AGE     Max age in seconds for active session detection
                        (default: 300)
